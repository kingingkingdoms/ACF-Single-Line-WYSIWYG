== Description ==

Add support for allowing the Advanced Custom Fields Pro Plugin to have a single line WYSIWYG editor field created by the Founder & CEO of the Millionaire's Digest with special thanks and all credits going to Tyler Bruffy for the original plugin.

== Installation ==

1. Visit Dashboard-> Plugins-> Add New
2. Install plugin
3. Activate
